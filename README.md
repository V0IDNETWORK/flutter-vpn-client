# Flutter VPN Client

A modern, open-source, cross-platform VPN client built with Flutter.

> 🚧 **This project is currently under active development.**

---

## About

Flutter VPN Client is an open-source project that aims to build a modern, fast, and user-friendly VPN application capable of running on multiple platforms from a single codebase.

The goal is not only to create a beautiful interface, but also to develop a complete VPN ecosystem including a scalable backend, secure authentication, server management, and seamless VPN connectivity using modern technologies.

The project is being developed publicly on GitHub, allowing anyone to follow its progress, report issues, suggest improvements, or contribute to the development.

---

## Vision

The long-term vision of this project is to provide a reliable, lightweight, and fully cross-platform VPN client with a clean architecture and maintainable codebase.

When completed, the application will support:

- Android
- iOS
- Windows
- macOS
- Linux
- Web *(where supported)*

---

## Technology Stack

### Frontend

- Flutter
- Dart

The entire user interface is built with Flutter to ensure a consistent experience across all supported platforms.

### Backend

- Node.js

The backend is responsible for:

- Authentication
- User management
- Subscription management
- VPN server management
- Configuration generation
- API services

### Database

- PostgreSQL

Used for securely storing:

- User accounts
- Subscription information
- VPN servers
- Connection logs
- Application data

---

## Supported VPN Technologies

The VPN core will be based on:

- V2Ray
- Xray

The application is designed to manage and connect to servers using these technologies while providing a simple and intuitive user experience.
